我的golang的包管理方法

现有的工具

  • dep
  • glide
  • govendor

问题

  • 墙外包
  • 项目路径有大小写，但是引用的包里面用的不一致，导致问题
  • git ssh拉包
  • 登录翻墙vpn+和登录公司内网vpn冲突
  • ss不支持终端

现在的方法

  • 使用https://github.com/godcong/glide版本的glide，对mirror进行了修复和增强

  • 添加mirrors.yaml到~/.glide/mirrors.yaml

更新mirror

上面的mirror总是一部分，如果你也有一个墙外包，可以按照下面的方法添加：

  • 去 https://gowalker.org 搜索该包
  • 如 https://google.golang.org/appengine/urlfetch对应https://gowalker.org/google.golang.org/appengine/urlfetch
  • 向上级点开，知道页面出现显示 README 内容
  • 点开并搜索 <some readme> site:github.com: https://www.google.com/search?q=Go+App+Engine+packages+site%3Agithub.com&oq=Go+App+Engine+packages+site%3Agithub.com&aqs=chrome..69i57j69i64l3.5944j0j1&sourceid=chrome&ie=UTF-8
  • 出现的结果第一个就是https://github.com/golang/appengine
